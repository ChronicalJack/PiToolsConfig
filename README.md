# PiToolsConfig
Raspberry Pi setup scripts

#Installation Code

To install this file onto your terminal, copy the following command into your terminal:

sudo apt update
sudo apt install git -y

cd ~
git clone https://github.com/ChronicalJack/PiToolsConfig

cd PiToolsConfig
chmod +x setup_pi_tools.sh
./setup_pi_tools.sh

source ~/.bashrc
